# myZemax - Optical Constructor

Графический интерфейс для оптического конструктора с функцией трассировки лучей (ray tracing).

## Описание

myZemax - это приложение для создания и визуализации оптических систем. Оно позволяет:
- Создавать различные оптические объекты (сферы, пирамиды, призмы, диски, плоскости)
- Настраивать их свойства (коэффициент преломления, отражения, цвет, позиция, размеры)
- Визуализировать сцену через ray tracing
- Управлять камерой для просмотра сцены
- Редактировать объекты через графический интерфейс

## Структура проекта

```
myZemax/
├── standard/          # Стандарт интерфейса (submodule)
├── include/           # Заголовочные файлы
│   ├── raytracer/    # Движок ray tracing
│   └── ui/           # Компоненты GUI
├── src/              # Исходные файлы
│   ├── raytracer/    # Реализации ray tracing
│   └── ui/           # Реализации UI
└── CMakeLists.txt    # Файл сборки

```

## Требования

- CMake 3.10 или выше
- C++17 компилятор
- Стандарт интерфейса (подключен как submodule)

## Сборка

```bash
mkdir build
cd build
cmake ..
make
```

## Установка плагинов

Перед запуском приложения необходимо поместить плагины в папку `plugins/`:

1. Создайте папку `plugins/backend/` (если её нет)
2. Поместите плагин бэкенда (например, `sfml_backend.so`) в `plugins/backend/`
3. Запустите приложение

**Важно:** Без плагина бэкенда приложение не запустится!

Подробные инструкции см. в [PLUGINS.md](PLUGINS.md).

## Использование

### Управление камерой

- **WASD** - движение камеры (W/S - вперед/назад, A/D - влево/вправо)
- **Space/Shift** - движение вверх/вниз
- **Стрелки** - поворот камеры

### Работа с объектами

- Клик левой кнопкой мыши по объекту в ray tracer - выбор объекта
- Клик по объекту в списке объектов - выбор и редактирование
- Редактирование свойств в окне Properties

### Панели

- **Control Panel** - управление камерой (кнопки)
- **Objects Panel** - список всех объектов (раскрывающаяся панель)
- **Properties Window** - редактирование свойств выбранного объекта
- **Toolbar** - меню плагинов (структура готова, плагины будут добавлены позже)

## Особенности

- Система плагинов для расширения функциональности
- Поддержка различных типов оптических объектов
- Интерактивное редактирование свойств объектов
- Визуальное выделение выбранных объектов
- Управление камерой с плавным ускорением
